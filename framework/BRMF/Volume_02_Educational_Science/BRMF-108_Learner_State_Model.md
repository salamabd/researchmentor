# BRMF-108 — Learner State Model

Status: Draft 1.0
Classification: Educational Science

## Purpose

This document defines the Learner State Model used by Business Research Mentor
(BRM). The learner state is BRM's educational representation of a student's
current learning condition at a point in time. It is designed to support
adaptive mentoring while remaining transparent and explainable.

## Guiding Principles

- The learner state is dynamic and continuously updated.
- It represents educational progress, not personal worth.
- It supports mentoring decisions but does not replace academic judgement.
- Every state change should be traceable and explainable.

## Core State Components

BRM maintains a structured view of the learner including:

- Scholarly development level
- Research judgement profile
- Cognitive pathway
- Current research stage
- Strengths
- Growth opportunities
- Confidence indicators
- Uncertainty indicators
- Intervention history
- Reflection history
- Supervisor interactions

## State Transitions

The learner state changes in response to:

- meaningful learner interactions;
- completed mentoring interventions;
- demonstrated reasoning;
- reflective activities;
- supervisor feedback.

## Educational Use

The learner state is used to:

- identify the next educational priority;
- select appropriate mentoring interventions;
- monitor progress over time;
- explain why recommendations are made;
- support meaningful discussions with supervisors.

## Explainability

Every recommendation should reference the learner state components that
contributed to the decision, ensuring transparency and educational integrity.

## Relationship to Previous Models

The Learner State Model integrates:

- BRMF-102 Scholarly Development Model
- BRMF-103 Research Judgement Model
- BRMF-104 Adaptive Mentoring Model
- BRMF-106 Cognitive Pathways Model
- BRMF-107 Mentoring Intervention Model

## Future Direction

Future versions may incorporate longitudinal learning analytics and predictive
models while preserving explainability and human oversight.
